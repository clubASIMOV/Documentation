### Les emphases

****

```
* Ce texte sera en italique *
_ Ce texte sera également en italique _

** Ce texte sera en gras **
__ Ce texte sera également en gras __

_ Vous **pouvez** bien sur les combiner _

```

```

  * Item 2a
  * Item 2b

```

****

#### Non ordonnées


```
1. Item 1
2. Item 2
3. Item 3
   1. Item 3a
   2. Item 3b
